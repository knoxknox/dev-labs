# gradle

Build:
```
$ gradle run - start application
$ gradle build - build gradle project
$ gradle bundle - copy jar dependencies
$ gradle -t run - start continuous build
```

Install:
```
$ sudo mkdir /opt/gradle
$ export PATH=$PATH:/opt/gradle/gradle-4.9/bin
$ wget https://services.gradle.org/distributions/gradle-4.9-bin.zip
$ sudo unzip -d /opt/gradle gradle-4.9-bin.zip && rm gradle-4.9-bin.zip
```
